## ルーティングについて

役割ごとにファイルを分けて行う。
`src/routes`にルーティングファイルを格納。

* `api.router.php` : API関連のルーティング
* `management.router.php` : 管理側のルーティング
* `web.router.php` : Webアプリ側のルーティング
